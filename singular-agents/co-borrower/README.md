# Loan Approval AI - Production System

AI-powered loan approval system using **Google Gemini + LangChain** for document processing and financial analysis.

## Features

✅ **Multi-Document Processing**
- ITR extraction (2 years)
- Bank statement analysis (6 months)
- Salary slip extraction (3 months)
- Parallel processing for speed

✅ **AI-Powered Extraction**
- Uses Gemini Vision for PDF analysis
- Structured output with Pydantic
- High accuracy with cross-validation

✅ **Financial Analysis**
- FOIR (Fixed Obligation to Income Ratio) calculation
- CIBIL score estimation
- Student loan eligibility assessment

✅ **Production Ready**
- FastAPI REST API
- Comprehensive error handling
- Logging and monitoring
- Async background tasks
